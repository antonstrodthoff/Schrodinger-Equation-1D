readdata.read_xyformat(filename)¶

Return type:nparray

readdata.save_nxyformat(filename, xx, yy)¶

can be used to save an array of format NxM into a file of N-X-Y-format

Parameters:
  • filename (string) – name of the file to be saved
  • xx (nparray) – the x values to be stored
  • yy (nparray) – multiple columns of y values to be stored
readdata.save_xyformat(filename, xx, yy)¶

  • xx (nparray) – the x values to be stored
  • yy (nparray) – the y values to be stored

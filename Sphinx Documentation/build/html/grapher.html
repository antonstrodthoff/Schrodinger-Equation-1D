grapher.plot(scalingfactor, changerange, axisrange)¶

reads the saved files and plots the information

Parameters:
  • scalingfactor (float) – amount the eigenstates will be scaled by (for better results)
  • changerange (boolean) – if the axis range will be changed or left at the default
  • axisrange ([float]) – has to be a list with 4 entries to specify xmin, xmax, ymin and ymax

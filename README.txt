Schroedinger Equation Solver in 1D
=================================

This program can solve the SE for any one dimensional, user specified problem.


Getting Started
---------------

To calculate and save the results the file main.py has to be executed with python via the terminal.
To plot the results the file plot.py has to be executed.

Requirements:
	-python
	-numpy
	-scipy
	-matplotlib
	-pytest
